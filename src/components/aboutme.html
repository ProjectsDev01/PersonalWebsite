My adventure with programming began back in high school, but until my high school years, my knowledge was mainly focused on simple console applications written in C++. It was only during my studies that I started to explore new technologies, such as those listed below

My Stack

C/C++ (Embedded)

I had the most experience with the C language, as it was the first one I learned. During the first stage of my studies, which I completed in the field of Industrial Automation, I took numerous courses such as basics of microprocessor technology and fundamentals of C and C++. Through these courses, I gained knowledge in programming processors and writing programs for the Atmega32 processor to control servo drives or stepper motors. Additionally, on my GitHub, there is a project for an acoustic equalizer written in C and Matlab for signal control and attenuation of specific sound frequencies using filters.

Java with Swing

Decided to learn Java along with the Swing library for the purpose of writing my engineering thesis. The goal was to create a GUI application that allows for the calculation of the minimal function using the Quine-McCluskey method for sequential automata. In the future, the project may be expanded with additional methods and could be released on a website.

Web Developer Tools

I started learning web development (both front-end and back-end) a year ago on the Udemy platform. I had previously learned HTML and CSS for my own needs in creating a simple CV. When I gained knowledge of creating APIs through the courses, I thought that programming the back-end of websites brought me a lot of satisfaction. During the course, I became familiar with technologies such as Express.js, Node.js with the Nodemon extension, React.js, and jQuery, as well as topics related to databases (PostgreSQL). Additionally, I covered several lessons related to Web3 and Blockchain.

Git, Github, Bash

As part of the same course, I gained basic knowledge of version control using the Bash terminal in Visual Studio Code, basic Linux commands, and later in the course, I solidified my knowledge of working on the Ubuntu system. As a result, a repository was created on GitHub, which I strive to keep updated and develop continuously.

Game Developer Tools

The latest field in which I decided to enhance my skills is Game Development and the Unity environment. Along with a few friends, we decided to create small projects that we hope will be developed into full-fledged games in the future. For these projects, I familiarized myself with the syntax of the C# language used in the Unity environment. We hope that the first game will be released later this year!

Other...
